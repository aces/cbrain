Comprendre les Projets

En sélectionnant le menu déroulant Créer un projet sous l'onglet Projets, vous pouvez créer un nouveau projet. Le projet qui en résulte est un projet de travail auquel seul le créateur du projet peut accéder. Des utilisateurs supplémentaires peuvent être ajoutés à un projet par l'administrateur système.

Projets

Les projets facilitent le partage de fichiers et de ressources entre les utilisateurs. Lorsqu'un Projet Actif est sélectionné, les fichiers et ressources associés à ce projet en particulier deviennent disponibles sous les différents onglets.

Il existe quatre types de projets : projets système, projets de site, projets utilisateur et projets de travail.
  • Projets Système
    • Chaque utilisateur sur CBRAIN est automatiquement membre du projet système "everyone".
  • Projets de Site
    • Chaque site possède un projet de site qui lui est automatiquement associé.
  • Projets Utilisateur
    • Chaque utilisateur sur CBRAIN est automatiquement assigné à un projet, dont il est le seul membre, identifié par son nom d'utilisateur.
  • Projets de Travail
    • Un projet de travail est un type de projet qui peut être créé par n'importe qui.
En haut de la page, le Projet Actif peut être sélectionné.
